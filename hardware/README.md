# hardware

3D-printable frame parts for the quad.

## My models

My own FreeCAD frame models. Each ships as an editable FreeCAD source
(`.FCStd`) and a printable mesh (`.3mf`):

- `drone_base` — the X-frame base that holds the four motors and the ESP32.
- `screw` — the fastener used to mount the parts.

[../LICENSE](../LICENSE)).
